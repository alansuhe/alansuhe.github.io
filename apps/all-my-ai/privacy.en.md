"All My AI" (hereinafter referred to as "the App") respects and protects user privacy. The App is a purely local tool—all data is stored only on the user's device locally and is not uploaded to any server.

## 1. Data Storage Scope

The App stores only the following information locally:
- Index of AI platform conversation records (for quick retrieval)
- User configuration information (such as proxy settings, account configurations, etc.)
- App preference settings (such as language, theme, etc.)

**The above data is stored only on the user's local device—not uploaded, not synced, not backed up.**

## 2. Data Collection Statement

The App **does not collect** any personal information, including but not limited to:
- Does not collect user identity information
- Does not collect device identifiers
- Does not collect usage behavior data
- Does not collect network activity records

## 3. External Service Interaction

The App interacts with external services in the following scenarios, but only when initiated by the user:

- **AI Platform Access**: When the user clicks to open a conversation, the browser accesses the corresponding AI platform. This interaction is handled by the browser; the App does not intervene.
- **AI Agent Launch**: When the user clicks to open an Agent, the App invokes a local terminal or editor to launch the corresponding tool. This operation is completed locally.

The App itself does not initiate any network requests and does not connect to any remote servers.

## 4. Data Security

- All data is stored within the App's sandbox directory, protected by macOS system
- Users can export or delete data through in-App functionality
- When uninstalling the App, all local data will be deleted along with it

## 5. Update Notice

This Privacy Policy may be updated as the App's functionality evolves. Updated policies will be published on the App's page.
